Linux以指定用户身份执行命令的两种方法

Author: cevin Date: 十二月 21, 2014  20:14:38 Category: linux

个人比较喜欢和推荐的一个方案

使用sudo

举个栗子：

cd /data0/www/s0 && sudo -u www-data git pull && sudo -u www-data /data0/tools/publish .

这个命令会在切进/data0/www/s0目录后以www-data用户的身份执行git代码拉取和发布操作，不用担心root身份权限混乱的问题

另外一个方案是依赖su

举个栗子：

cd /data0/www/s0 && su - www-dat -c "git pull && /data0/tools/publish ."
